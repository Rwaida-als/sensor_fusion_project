Sensor Fusion Module using a Bayesian Network

Team Members

  • Al Waleed Al Saleemi
  • Rwaida Alssadi

Project Description

This project involves developing a sensor fusion module based on a Bayesian Network to help the robot determine if it passed a door and its distance from the wall. The data for the Bayesian Network is obtained from iCreate robots, and the CPTs are learned from measurements.
